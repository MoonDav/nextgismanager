What is wxGIS?

wxGIS is OpenSource GIS written with C++ and wxWidgets and GDAL libraries. wxGIS is following ArcGIS ideology in developing GIS. Now wxGIS includes:

  • wxGISCatalog - ArcCatalog analogue.
  • wxGISToolbox - ArcToolbox analogue.
